# 심화 국어 기말 학습 사이트

2026학년도 1학기 「심화 국어」 기말고사(7/3) 대비 단일 페이지 학습 앱입니다.
한국사 학습 앱 `myhistorical`의 구현 방식을 참고하여 새로 제작했습니다.

## 기능
- **작품 학습**: 영역(고전 시가·현대시·고전 산문·독서)별 작품 카드 — 원문/해제/주제/화자·정서/표현상 특징/시어 풀이, 작품별 플래시카드
- **모의시험**: 유형(객관식·서답형·혼합)·영역·난이도·문항 수 설정 → 채점 → 오답 다시 풀기
- **O/X 퀴즈**: 영역·문항 수 설정 → 즉시 해설

## 사용
`index.html`을 브라우저로 열면 됩니다. 라이트/다크 테마, 진도 저장(localStorage), 모바일 PWA 설치를 지원합니다.

## 출처
EBS 수능특강(2027) 시험범위 기반. 고전 시가·고전 산문은 원전(공유저작물), 독서 4지문은 원문 대신 요약·구성으로 정리했습니다. 해제·주제·표현 등 분석은 자체 작성입니다.
